🏥 Sistema

GET /

Retorna información general sobre la API y sus endpoints disponibles

Respuesta de ejemplo:

{
  "success": true,
  "message": "API de Sistema de Productos AIMEC",
  "version": "1.0.0",
  "documentation": "/api-docs",
  "endpoints": {
    "health": "/health",
    "products": "/api/products",
    "categories": "/api/categories",
    "specifications": "/api/specifications"
  }
}
GET /health
Estado del Sistema
Verifica el estado de salud del sistema y la conexión a la base de datos

{

📦 Productos

GET /api/products
Listar Productos
Obtiene una lista de productos con filtros opcionales
GET /api/products/search
Buscar Productos
Busca productos por nombre o descripción
GET /api/products/featured
Productos Destacados
Obtiene productos marcados como destacados
GET /api/products/{id}
Obtener Producto por ID
Obtiene un producto específico por su ID
POST /api/products
Crear Producto
Crea un nuevo producto
PUT /api/products/{id}
Actualizar Producto
Actualiza un producto existente
DELETE /api/products/{id}
Eliminar Producto
Elimina un producto por su ID

📂 Categorías

GET /api/categories
Listar Categorías
Obtiene una lista de todas las categorías
GET /api/categories/{id}
Obtener Categoría
Obtiene una categoría específica por su ID
POST /api/categories
Crear Categoría
Crea una nueva categoría

🔧 Especificaciones

GET /api/specifications/types
Tipos de Especificaciones
Obtiene los tipos de especificaciones disponibles
GET /api/specifications/products/{productId}
Especificaciones de Producto
Obtiene las especificaciones de un producto específico
